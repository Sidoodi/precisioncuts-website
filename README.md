# Precision Cuts NYC Static Site

This repository contains the static HTML for the Precision Cuts NYC landing page.

## Serving locally

Click to open the page directly in your browser: [Open `index.html`](./index.html).

If you prefer a local server, use Python's built-in HTTP server from the repository root:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) to preview the page.
