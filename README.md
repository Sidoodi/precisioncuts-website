# Precision Cuts NYC website

This repository contains the single-page static site for Precision Cuts NYC.

## View locally
Serve the HTML file with any static server and open it in your browser:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000/precisioncuts-website](http://localhost:8000/precisioncuts-website) to view the page.

If you prefer to open the file directly without a server, you can also use the `file://` protocol.
